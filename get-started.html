    This page is for instructions on how to install HiyaCFW to a stock DSi. (meaning a DSi that does not already have CFW on it.)

    Different system firmware versions will require different steps to be able to get HiyaCFW! This page will help you find where to start for your DSi's firmware.

    The pictures in this guide can be enlarged to see clearer by clicking on them.

    Before starting, Windows users should enable the option to show file extensions using this guide.

    Before starting, it's a good idea to check your SD card for errors using H2testw (Windows), F3 (Linux), or F3X (Mac).

    This will not work with Chinese DSi systems (A.K.A. iQue DSi) or Korean DSi systems as none of the exploitable apps have been released there.

    Select the appropriate page for your version from the chart below.

    Your system firmware version can be found at the bottom right of the top screen of the System Settings.
